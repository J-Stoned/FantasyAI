Yahoo! Fantasy Console

This is a website that leverages the Yahoo! Fantasy Sports Wrapper Nuget Package. This package is a .NET Standard 2.0 Wrapper for the Yahoo Fantasy Sports API.

The API is designed to act as a helper for those interacting with the Y! Fantasy API. The goal is to get acquainted with the Api and show how the wrapper can help with the exisitng painpoints with the Api.

Setup

Login to this app (your username and password is not saved, all that this collects is your public Yahoo! profile and a token to make subsequent Api calls)

For more information on this package, take a look at the public GitHub Repo.
